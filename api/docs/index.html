Getting started

In order to get started with the API usage you have to login and generate your own SESSION. After loggin in you can use the entire functionallity of the school administration software.

Logging in

To log your self in you have to call the page API_login.php and transmit via POST request the username and the password in plain-text. As a response you will either get a success or failure file.
